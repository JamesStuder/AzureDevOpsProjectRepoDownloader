# Azure DevOps Project Repo Downloader
Downloads all repos in an Azure DevOps Project.  If projects have already been downloaded then a pull will be performed, otherwise a clone will be done.

# How To:
1. Clone
2. Build
3. Run
   1. Answer Questions:
      1. Input the url to your DevOps Environment
      2. Input the project name
      3. Input your PAT
      4. Input the root location where you want the repos downloaded too
